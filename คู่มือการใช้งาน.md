## 💻 การใช้งานระบบ

### สำหรับอาจารย์

#### 1. เข้าสู่ระบบ
- ใช้บัญชีที่ได้รับจากผู้ดูแลระบบ
- กดปุ่ม "เข้าสู่ระบบ"

#### 2. สร้างวิชาใหม่
- คลิกปุ่ม "+ เพิ่มวิชาใหม่"
- กรอกข้อมูลวิชา:
  - **รหัสวิชา**: เช่น CS101
  - **ชื่อวิชา**: เช่น Computer Programming
  - **เวลาเข้าเรียน**: เช่น 09:00
  - **ขีดจำกัดเวลาสาย**: เช่น 15 (นาที)
  - **ขีดจำกัดเวลาขาด**: เช่น 30 (นาที)

#### 3. ตั้งค่าการเช็คชื่อ
ระบบจะคำนวณสถานะดังนี้:
- **มา**: เช็คชื่อก่อนหรือตรงเวลาเรียน
- **สาย**: เช็คชื่อหลังเวลาเรียนแต่ไม่เกินที่กำหนด
- **ขาด**: เช็คชื่อเกินเวลาที่กำหนด

**ตัวอย่าง:**
- เวลาเรียน: 09:00
- สายสุด: 15 นาที (เช็คชื่อก่อน 09:15 จะได้สถานะ "สาย")
- ขาดเมื่อเกิน: 30 นาที (เช็คชื่อหลัง 09:30 จะได้สถานะ "ขาด")

#### 4. เช็คชื่อแบบเรียลไทม์
- เลือกวิชาที่ต้องการ
- คลิก "ดูการเช็คชื่อ"
- ให้นักเรียนแสกน RFID Card
- ระบบจะแสดงผลทันที:
  - ชื่อนักเรียน
  - สถานะ (มา/สาย/ขาด)
  - เวลาที่เช็คชื่อ

#### 5. ดูสถิติการเข้าเรียน
- ระบบจะแสดงจำนวนนักเรียนที่:
  - ✅ มา
  - ⏰ สาย
  - ❌ ขาด
- อัปเดตอัตโนมัติทุก 5 วินาที

## 📡 การเชื่อมต่อ RFID Reader

### วิธีที่ 1: ใช้ RFID Reader แบบ USB (แนะนำ)

RFID Reader แบบ USB จะทำงานเหมือน Keyboard:

1. เสียบ RFID Reader เข้ากับระบบอุปกรณ์ที่ส่งข้อมูลเข้า API
2. ตั้งวิชาปัจจุบันในหน้า Dashboard
3. ให้นักเรียนแสกนการ์ดผ่านอุปกรณ์จริง
4. ระบบจะบันทึกและแสดงผลในหน้าเช็คชื่ออัตโนมัติ

### วิธีที่ 2: เชื่อมต่อผ่าน API (สำหรับ RFID Reader ที่มีซอฟต์แวร์เฉพาะ)

หาก RFID Reader ของคุณมีซอฟต์แวร์หรือสคริปต์ที่สามารถส่งข้อมูลผ่าน HTTP ได้:

**API Endpoint:**
```
POST http://localhost:3000/api/attendance/scan
Content-Type: application/json

Body:
{
  "rfid": "9ef13a02",
  "courseId": 1
}
```

**ตัวอย่างการใช้งานด้วย Python:**
```python
import requests

def scan_rfid(rfid_code, course_id):
    url = "http://localhost:3000/api/attendance/scan"
    data = {
        "rfid": rfid_code,
        "courseId": course_id
    }
    response = requests.post(url, json=data)
    print(response.json())

# เมื่อได้รหัส RFID จาก Reader
scan_rfid("9ef13a02", 1)
```

**ตัวอย่างการใช้งานด้วย Arduino/ESP32:**
```cpp
#include <WiFi.h>
#include <HTTPClient.h>

void scanRFID(String rfidCode, int courseId) {
  HTTPClient http;
  http.begin("http://192.168.1.100:3000/api/attendance/scan");
  http.addHeader("Content-Type", "application/json");
  
  String payload = "{\"rfid\":\"" + rfidCode + "\",\"courseId\":" + String(courseId) + "}";
  int httpCode = http.POST(payload);
  
  if (httpCode > 0) {
    String response = http.getString();
    Serial.println(response);
  }
  http.end();
}
```

## ❓ คำถามที่พบบ่อย

### Q: นักเรียนแสกนการ์ดแล้วแต่ไม่มีข้อมูล?
**A:** ตรวจสอบว่าได้ลงทะเบียน RFID Card ของนักเรียนในระบบแล้วหรือยัง

### Q: ระบบคำนวณเวลาไม่ถูกต้อง?
**A:** ตรวจสอบการตั้งค่าเวลาในคอมพิวเตอร์และ timezone

### Q: ต้องการเปลี่ยนรหัสผ่านอาจารย์ยังไง?
**A:** ใช้ bcrypt เพื่อ hash รหัสผ่านใหม่และอัปเดตในฐานข้อมูล:
```javascript
const bcrypt = require('bcryptjs');
const newPassword = bcrypt.hashSync('รหัสผ่านใหม่', 10);
console.log(newPassword);
```

### Q: สามารถใช้งานได้หลายคอมพิวเตอร์พร้อมกันไหม?
**A:** ได้ครับ แต่ต้องเปลี่ยน `localhost` เป็น IP Address ของเครื่องที่รันเซิร์ฟเวอร์

**ตัวอย่าง:**
- เครื่อง Server: IP 192.168.1.100 รัน `npm start`
- เครื่องอื่นๆ: เปิดเว็บที่ `http://192.168.1.100:3000`

### Q: ข้อมูลเก็บไว้ที่ไหน?
**A:** ข้อมูลเก็บในไฟล์ `attendance.db` (SQLite Database) ในโฟลเดอร์เดียวกับโปรเจกต์

### Q: ต้องการ export ข้อมูลเป็น Excel ยังไง?
**A:** สามารถเพิ่มฟีเจอร์ export โดยใช้ library เช่น `xlsx` หรือใช้โปรแกรม SQLite Browser เพื่อดูและ export ข้อมูล
