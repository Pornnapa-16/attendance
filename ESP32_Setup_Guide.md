# 🔧 คู่มือการติดตั้งและใช้งาน ESP32 + RFID

## 📦 อุปกรณ์ที่ต้องใช้

### **บอร์ดและโมดูลหลัก**
1. **ESP32 Development Board** (NodeMCU-32S หรือรุ่นอื่น)
2. **RFID Reader Module RC522** (13.56MHz)
3. **สาย Jumper Wire** สำหรับต่อวงจร
4. **Breadboard** (ถ้าต้องการ)

### **อุปกรณ์เสริม (Optional)**
5. **LED 3 สี** (เขียว สีเหลือง แดง) สำหรับแสดงสถานะ
6. **Buzzer** สำหรับเสียงแจ้งเตือน
7. **ตัวต้านทาน 220Ω** สำหรับ LED (3 ตัว)

---

## 🔌 การต่อสาย (Wiring Diagram)

### **การต่อ RFID-RC522 กับ ESP32** (✅ สำคัญมาก!)

| RFID RC522 Pin | ESP32 Pin | สาย       |
|---------------|-----------|-----------|
| **SDA (SS)**  | GPIO 5    | สีเหลือง   |
| **SCK**       | GPIO 18   | สีเขียว    |
| **MOSI**      | GPIO 23   | สีน้ำเงิน  |
| **MISO**      | GPIO 19   | สีม่วง     |
| **IRQ**       | ไม่ต่อ     | -         |
| **GND**       | GND       | สีดำ      |
| **RST**       | GPIO 22   | สีขาว     |
| **3.3V**      | 3.3V      | สีแดง     |

### **การต่อ LED และ Buzzer (Optional)**

| อุปกรณ์         | ขา +        | ขา -  | หมายเหตุ           |
|----------------|-------------|-------|-------------------|
| **LED สีเขียว** | GPIO 2      | GND   | ผ่านตัวต้านทาน 220Ω |
| **LED สีเหลือง**| GPIO 4      | GND   | ผ่านตัวต้านทาน 220Ω |
| **LED สีแดง**   | GPIO 15     | GND   | ผ่านตัวต้านทาน 220Ω |
| **Buzzer**     | GPIO 16     | GND   | -                 |

## 💻 การติดตั้ง Arduino IDE และ Library

### **ขั้นตอนที่ 1: ติดตั้ง Arduino IDE**
1. ดาวน์โหลด [Arduino IDE](https://www.arduino.cc/en/software) เวอร์ชันล่าสุด
2. ติดตั้งและเปิดโปรแกรม

### **ขั้นตอนที่ 2: เพิ่ม ESP32 Board Manager**
1. เปิด Arduino IDE
2. ไปที่ **File** → **Preferences**
3. ในช่อง **Additional Board Manager URLs** วาง URL นี้:
   ```
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   ```
4. คลิก **OK**
5. ไปที่ **Tools** → **Board** → **Board Manager**
6. ค้นหา "**ESP32**" และคลิก **Install**

### **ขั้นตอนที่ 3: ติดตั้ง Libraries**
ไปที่ **Tools** → **Manage Libraries** แล้วค้นหาและติดตั้ง:

1. **MFRC522** (by GithubCommunity)
2. **ArduinoJson** (by Benoit Blanchon) - เลือกเวอร์ชัน 6.x

## ⚙️ การตั้งค่าโค้ด ESP32

### **1. แก้ไขข้อมูล WiFi**
เปิดไฟล์ `ESP32_RFID_Reader.ino` และแก้ไขบรรทัดเหล่านี้:

```cpp
const char* WIFI_SSID = "YOUR_WIFI_NAME";        // ชื่อ WiFi ของคุณ
const char* WIFI_PASSWORD = "YOUR_WIFI_PASSWORD"; // รหัส WiFi ของคุณ
```

**ตัวอย่าง:**
```cpp
const char* WIFI_SSID = "MyHome_WiFi";
const char* WIFI_PASSWORD = "12345678";
```

### **2. แก้ไข IP Address ของ Server**
หา IP Address ของคอมพิวเตอร์ที่รัน server:

**Windows PowerShell:**
```powershell
ipconfig
```
หา **IPv4 Address** ที่ขึ้นต้นด้วย `192.168.x.x`

**แก้ไขโค้ด:**
```cpp
const char* SERVER_URL = "http://192.168.1.100:3000/api/attendance/scan";
//                              ^^^^^^^^^^^^^^^^ เปลี่ยนเป็น IP ของคุณ
```

**ตัวอย่าง:**
```cpp
const char* SERVER_URL = "http://192.168.1.45:3000/api/attendance/scan";
```

### **3. ตั้งค่า Course ID**
เปลี่ยน `COURSE_ID` ให้ตรงกับวิชาที่ต้องการเช็คชื่อ:

```cpp
const int COURSE_ID = 1; // ID ของวิชา (ดูได้จากเว็บ Dashboard)
```

## 📤 การอัปโหลดโค้ดลง ESP32

### **ขั้นตอน:**
1. เสียบ ESP32 เข้ากับคอมพิวเตอร์ผ่านสาย USB
2. เปิด Arduino IDE
3. เลือกบอร์ด: **Tools** → **Board** → **ESP32 Dev Module**
4. เลือก Port: **Tools** → **Port** → เลือก COM Port ที่ ESP32 เสียบอยู่
5. คลิก **Upload** (ปุ่มลูกศรขวา)
6. รอจนกว่าจะขึ้น "**Done uploading**"

## 🚀 การใช้งาน

### **ขั้นตอนการเริ่มต้น:**

1. **เปิด Server**
   ```powershell
   cd C:\Users\Pornnapa\attendance
   node server.js
   ```

2. **เสียบ ESP32 เข้ากับแหล่งจ่ายไฟ** (USB หรือ Adapter 5V)

3. **เปิด Serial Monitor** (Arduino IDE)
   - ไปที่ **Tools** → **Serial Monitor**
   - ตั้งค่า Baud Rate เป็น **115200**

4. **ตรวจสอบการเชื่อมต่อ**
   ใน Serial Monitor จะแสดง:
   ```

     ระบบเช็คชื่อด้วย RFID + ESP32
   =================================

   ......
   ✅ เชื่อมต่อ WiFi สำเร็จ!
   📡 IP Address: 192.168.1.XXX
   
   ✅ ระบบพร้อมใช้งาน!
   📖 วางบัตร RFID บนเครื่องอ่าน...
   ```

5. **วางบัตร RFID** บนเครื่องอ่าน RC522
   - ระบบจะอ่าน RFID UID
   - ส่งข้อมูลไปที่ Server
   - แสดงผลลัพธ์และสถานะ

## ❗ การแก้ไขปัญหา (Troubleshooting)

### **1. ESP32 เชื่อมต่อ WiFi ไม่ได้**
- ✅ ตรวจสอบชื่อ WiFi และรหัสผ่านให้ถูกต้อง
- ✅ ตรวจสอบว่า WiFi เป็น 2.4GHz (ESP32 ไม่รองรับ 5GHz)
- ✅ เข้าใกล้ Router ให้มากขึ้น

### **2. RFID อ่านไม่ได้**
- ✅ ตรวจสอบการต่อสายให้ถูกต้อง (โดยเฉพาะ SDA, SCK, MOSI, MISO)
- ✅ ตรวจสอบว่า RFID Module ได้ไฟ 3.3V
- ✅ วางบัตรให้ชิดกับเครื่องอ่าน (ห่างไม่เกิน 3-5 cm)
- ✅ ทดสอบด้วยโค้ด DumpInfo Example ของ Library MFRC522

### **3. ส่งข้อมูลไม่ถึง Server**
- ✅ ตรวจสอบว่า Server เปิดอยู่ (`node server.js`)
- ✅ ตรวจสอบ IP Address ให้ถูกต้อง
- ✅ ตรวจสอบว่าคอมพิวเตอร์และ ESP32 อยู่ WiFi เดียวกัน
- ✅ ปิด Firewall ชั่วคราวเพื่อทดสอบ

### **4. Error: "ไม่พบข้อมูลนักเรียน"**
- ✅ ตรวจสอบว่าได้ลงทะเบียน RFID ในฐานข้อมูลแล้ว
- ✅ ตรวจสอบ RFID UID ให้ตรงกับที่บันทึกไว้

### **5. Error: "นักเรียนเช็คชื่อแล้ววันนี้"**
- ✅ นักเรียนสามารถเช็คชื่อได้วันละ 1 ครั้ง
- ✅ ถ้าต้องการเช็คซ้ำ ให้ลบข้อมูลในฐานข้อมูลก่อน
